PZ Modsmith
===========

PZ Modsmith is a local helper for building and checking Project Zomboid dedicated-server mod lists.

Give it either:

- a Project Zomboid console log from a server join attempt, or
- a plain list of Steam Workshop IDs/URLs

It scans your local Steam Workshop downloads, reads each `mod.info`, and produces draft server config lines:

  WorkshopItems=...
  Mods=...

It is especially useful for Workshop items that contain more than one Mod ID (variants, patches, optional add-ons, libraries, legacy/B41 leftovers).

Important: PZ Modsmith reads local Workshop downloads only. It does not download mods from Steam.


Requirements
============

CLI mode:

  Python 3.10+ (standard library only)

Web UI mode:

  Flask is required.

  python3 -m venv .venv
  source .venv/bin/activate
  pip install flask


Running It
==========

From this directory:

  python3 pz_modsmith.py --help

or:

  python3 -m pz_modsmith --help


Web UI
======

Start the local browser UI:

  python3 pz_modsmith.py --web

Default bind:

  http://127.0.0.1:8765

Optional host and port:

  python3 pz_modsmith.py --web --host 127.0.0.1 --port 9000

In the UI, provide:

  1. Your Steam Workshop content path for Project Zomboid.
  2. A console.txt upload, pasted console text, Workshop IDs, or Steam URLs.
  3. Review selections for multi-Mod-ID Workshop items.
  4. Copy the generated WorkshopItems= and Mods= lines.

The web UI can also download a ZIP containing the same reports generated by CLI mode.


App Mode
========

Start PZ Modsmith in local app-style mode:

  python3 pz_modsmith.py --app

This runs the same local Flask UI as --web, but automatically opens your browser.

If the default port is busy, app mode tries the next available port automatically.


Workshop Path
=============

PZ Modsmith expects the Project Zomboid Workshop content folder, usually named 108600.

It checks these common Linux paths by default:

  /mnt/storage/SteamLibrary/steamapps/workshop/content/108600
  ~/.steam/debian-installation/steamapps/workshop/content/108600
  ~/.steam/steam/steamapps/workshop/content/108600
  ~/.local/share/Steam/steamapps/workshop/content/108600

If your Steam library lives somewhere else, pass --workshop-path or enter the path in the web UI.


How Selection Works
===================

For each Workshop ID, PZ Modsmith searches recursively for `mod.info` files.

If one unique Mod ID is found, it selects that Mod ID automatically.

If multiple Mod IDs are found, it ranks them with simple text heuristics and (when console logs are provided) boosts Mod IDs that the game actually loaded.

  Positive hints:
    B42, Build 42, MP, multiplayer, current, fixed, maintained

  Negative hints:
    B41, Build 41, SP, single-player, legacy, pre-42, pre-MP, disable,
    do-not-activate, optional, patch, compatibility, addon, no-MP, unstable,
    deprecated


Generated Reports
=================

CLI mode writes reports to ./pzmodsmith-output by default.

Reports include:

  - draft server lines (WorkshopItems= and Mods=)
  - normalized workshop id list
  - multi-mod review list
  - warnings and dependency hints
  - missing item list
  - structured JSON results


Important Notes
===============

Review the generated Mods= line before using it on a real server. Some Workshop items intentionally include several Mod IDs, patches, compatibility modules, or optional add-ons.

If a Workshop ID is listed as missing, make sure the item is subscribed, downloaded, and present under the 108600 Workshop content folder.
